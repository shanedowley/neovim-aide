# Neovim-AIDE Agent Instructions

Neovim-AIDE is a Neovim IDE built around the principle:

> Human-controlled. AI-assisted.

Before performing substantial analysis, planning or implementation work in
this repository, read the Neovim-AIDE AI Context Pack.

Read the documents in this order:

1. `docs/ai/README.md`
2. `docs/ai/PROJECT_CONTEXT.md`
3. `docs/ai/PRODUCT_PHILOSOPHY.md`
4. `docs/ai/DEVELOPMENT_GUIDE.md`
5. `docs/ai/AI_WORKING_AGREEMENT.md`
6. `docs/ai/ROADMAP.md`
7. `docs/ai/PROJECT_TERMINOLOGY.md`

These documents describe the project's intent, philosophy, development
practices, strategic direction and preferred terminology.

They complement—but never replace—the source code and technical
documentation.

If a conflict exists between these documents and the implementation,
the implementation is authoritative.

When contributing:

- seek understanding before implementation
- preserve existing architecture
- prefer small, focused and incremental changes
- avoid speculative refactoring
- validate meaningful changes
- update relevant documentation
- explain risks and trade-offs clearly
- remember that the AI assists and the developer decides
